## 图片规格建议
- 尺寸：400x300 像素（4:3比例）
- 格式：JPG 或 PNG
- 大小：每张不超过 100KB

## 获取图片建议
1. 使用维基百科、博物馆官网等公共领域图片
2. 确保图片版权清晰或使用授权图片
3. 选择最能代表该风格的经典作品

如果没有图片，系统会自动显示占位符图标。
